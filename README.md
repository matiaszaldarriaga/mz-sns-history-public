# IAS SNS History (Public)

Public home for the IAS School of Natural Sciences history dataset:
- report data problems
- leave comments
- download snapshot releases

## Quick Links
- [Report a data issue](https://github.com/matiaszaldarriaga/mz-sns-history-public/issues/new?template=data_correction.yml)
- [Leave a general comment](https://github.com/matiaszaldarriaga/mz-sns-history-public/issues/new?template=general_comment.yml)
- [All issues](https://github.com/matiaszaldarriaga/mz-sns-history-public/issues)
- [Releases / snapshots](https://github.com/matiaszaldarriaga/mz-sns-history-public/releases)

## What This Repo Is
This repository is the public feedback and release hub.

The working research codebase and internal tooling are maintained separately.

## What To Include In A Report
- scholar name (and ID if visible)
- what looks wrong
- what you think is correct
- source link (if available)

Thanks for helping improve the dataset.
